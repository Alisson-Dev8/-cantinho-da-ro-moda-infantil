# Cantinho da Ro - Moda Infantil

Landing page da loja infantil Cantinho da Ro, migrada de Vite+React para **Next.js 14** com App Router.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui** (Radix UI)
- **Lucide React**

## Rodando o projeto

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
